# Responsive Preview

Responsive Preview allows applications to be tested across multiple screen sizes.

Supported preview modes:

* Desktop
* Tablet
* Mobile

Users can validate:

* Layout responsiveness
* Component scaling
* Mobile behavior
* Adaptive UI structures

Responsive testing ensures applications work across devices.
