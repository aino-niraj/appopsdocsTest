# Editor Overview

The AppOps Builder Editor is the main visual development environment used to build applications.

The editor contains multiple sections that work together to simplify application development.

## Left Sidebar

Contains:

* Visual Elements
* Container Elements
* Form Elements
* Reusable Components

## Canvas Workspace

The central design area where pages and layouts are created visually.

## Right Configuration Panel

Used to configure:

* Styles
* Properties
* Dynamic Bindings
* Events

## Bottom Navigation

Switch between:

* Page
* Component
* Client Flow

## Top Toolbar

Provides:

* Responsive Preview
* Undo / Redo
* Refresh
* Layout Controls
