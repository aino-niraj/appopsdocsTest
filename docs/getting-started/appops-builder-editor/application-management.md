# Application Management

Application Management helps organize and maintain applications inside AppOps Builder.

Users can manage:

* Services
* Applications
* Components
* Themes
* Extensions
* Environments

## Services

Services group related applications and workflows.

## Applications

Applications contain pages, components, and user interfaces.

## Components

Reusable UI elements shared across applications.

## Environments

Manage deployment and runtime environments.

Application Management helps maintain scalable project organization.
