# Quick Start Guide

Welcome to the AppOps Builder Editor.

This guide helps new users understand the editor workspace and start building applications visually.

In this guide, you will learn how to:

* Navigate the editor
* Create pages
* Add components
* Configure styles
* Add events
* Preview applications

The editor is divided into three main sections:

* Component Panel
* Canvas Workspace
* Configuration Panel

Applications are built visually by dragging components into the canvas and configuring their behavior using the right-side panels.
