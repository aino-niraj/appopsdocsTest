# Config Tab

The Config tab is used to configure component properties and dynamic bindings.

Users can:

* Add properties
* Bind values
* Connect APIs
* Configure variables
* Use dynamic providers

## Properties

Each component supports configurable properties such as:

* text
* value
* placeholder
* src
* label

## Dynamic Binding

Properties can connect to:

* Variables
* APIs
* State Values
* Workflows

## Providers

Providers allow reusable and dynamic data configurations across the application.
