# Event Tab

The Event tab is used to create interactive application behavior.

Events allow components to respond to user actions.

## Supported Events

* click
* hover
* change
* submit
* focus

## Triggers

Triggers define when an action should execute.

## Methods

Methods define what action should happen after an event is triggered.

Examples:

* Navigate to Page
* Call API
* Show Popup
* Update State
* Submit Form

The event system enables dynamic and interactive applications without manually writing frontend logic.
