# Client Flow

Client Flow is used to visually manage frontend workflows and user interactions.

It helps define:

* Navigation flow
* User journeys
* Workflow execution
* State transitions

## Flow Builder

Applications flows can be visually connected and managed.

## Navigation Flow

Configure page-to-page navigation behavior.

## Workflow Integration

Connect UI events with workflows and backend logic.

Client Flow simplifies frontend interaction management.
