# Pages

Pages represent individual screens inside an application.

Examples:

* Home Page
* Dashboard
* Login Page
* Analytics Page

## Creating Pages

1. Open the Pages section
2. Click **+ New Page**
3. Enter the page name
4. Click **Confirm**

## Managing Pages

Users can:

* Rename pages
* Organize pages
* Configure navigation
* Manage layouts

Each page acts as an independent UI workspace.
